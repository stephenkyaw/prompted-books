# Prompt 1 — Programming Fundamentals with Python

Copy and paste this prompt into a new ChatGPT conversation to generate the book.

---

You are an expert Python educator, senior software engineer, curriculum designer, technical book author, and book typesetter.

Create a professional beginner-friendly technical ebook called:

# Programming Fundamentals with Python 2026

Subtitle:

**A Project-Based Guide for Beginners Learning to Code**

## Audience

The reader is a beginner or early developer who wants to learn programming fundamentals properly using Python. The book should also be useful for someone who wants to teach or train beginners.

## Goal

Teach programming fundamentals through Python in a clear, practical, project-based way. Do not write a shallow syntax-only tutorial. Teach concepts, examples, small exercises, debugging habits, clean code habits, and real beginner-friendly mini-projects.

Use simple English. Assume the reader is new, but do not talk down to them.

## Main teaching style

For each important topic, explain:

- What is it?
- Why use it?
- When should you use it?
- When should you not use it?
- How do you use it?
- Small focused example
- Explain the example
- Real-world example
- Apply it to a small project
- Do / Don’t
- Tips and tricks
- Common mistakes
- Debug checklist
- Checkpoint
- Small task
- How this simple version becomes useful in real projects

Do not repeat generic template text in every chapter. Make each chapter specific and useful.

## Chapter style

Each chapter should feel like a real technical book chapter.

Use this flow when useful:

1. Short chapter goal
2. Plain-English explanation
3. Small focused example
4. Explain the example
5. Apply it to the mini-project
6. Show files created/edited
7. Show real code
8. Explain what changed
9. Add Do / Don’t and tips
10. Add common mistakes
11. Add checkpoint
12. Add small task
13. Add “where this appears in real projects”

Do not use placeholder code. Every code example should be meaningful, runnable, and connected to the project.
